# koa2_study_project
koa2由于使用了更高的es版本，所有node版本在v7.6.0

##### 使用n来管理node版本
`sudo npm install -g n`

###### 常用命令
``` $n //查看已安装版本  
$n latest  //安装最新版本并使用  
$n latest -d   //下载最新版但不使用，-d参数表示为仅下载  
$n stable  //安装最新稳定版本并使用  
$n <version>  //安装某个版本并使用，如$n 6.2.2  
$n rm <version ...> //删除某些版本  
$n ls    //查看可用版本  
$n --latest    //查看最新版本  
$n --stable    //查看最新稳定版  
$n -h    //查看帮助信息，更多命令在这里查看 ```
